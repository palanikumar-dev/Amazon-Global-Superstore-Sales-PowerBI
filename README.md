# Amazon-Global-Superstore-Sales-PowerBI

**Objective**

* Analyze Amazon Global Superstore sales data to identify trends and opportunities.

* Build an interactive Power BI dashboard to support business decision-making.

* Provide actionable insights on top-performing products, regions, and sales channels.

**Scope of the Project**

**In-Scope:**

* Data cleaning and preprocessing in Excel.

* Analysis of sales, profit, and customer segments.

* Creating interactive Power BI dashboards with key metrics.

**Out-of-Scope:**

* Real-time sales tracking.

* Predictive modeling or AI-based sales forecasting.

**Tools Used**

**Excel:** 
Data cleaning, handling missing values, formatting.

**Power BI:** 
Data visualization, dashboard creation, interactive reporting.

**Data Cleaning (Excel)**

* Removed duplicate records to avoid skewed analysis.

* Standardized date formats and numerical fields.

* Handled missing or inconsistent values in columns like sales, profit, and region.

* Created calculated columns (e.g., Profit Margin = Profit / Sales).

* Filtered out irrelevant or incomplete records.

**Power BI Dashboard**

**Key Features:**

* Interactive filters by region, category, and year.

* KPIs: Total Sales, Profit, Quantity Sold, and Profit Margin.

* Visualizations: Bar charts for top products, line charts for sales trends, maps for regional performance.

* Drill-through functionality to analyze customer segments in detail.

* Monthly and yearly sales trend analysis.

**Outcomes**

* Identified top-selling products and underperforming regions.

* Analyzed monthly and yearly sales trends, enabling better inventory planning.

* Provided insights into profit margins by product category.

* Delivered an interactive Power BI dashboard that improved decision-making speed.
